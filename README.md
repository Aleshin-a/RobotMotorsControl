# RobotMotorsControl

Скрипт для управления двигателями робота.

## Описание

Этот скрипт позволяет управлять двигателями робота, обеспечивая движение вперед с заданной скоростью и остановку.  Имеется возможность плавного изменения скорости для более контролируемого движения.

## Функциональность

Скрипт предоставляет следующие функции:

*   **`move_forward(speed, duration)`**:  Движение вперед с заданной скоростью (`speed`) в течение заданного времени (`duration`).
*   **`stop()`**: Остановка двигателей.
*   **`set_speed(speed)` (только в версии с плавным изменением скорости)**: Устанавливает скорость двигателей.

## Описание функций

### `move_forward(speed, duration)`

*   **Описание:**  Обеспечивает движение вперед с заданной скоростью в течение указанного времени.
*   **Параметры:**
    *   `speed` (int):  Скорость движения (например, от 0 до 100).
    *   `duration` (int/float):  Время движения в секундах.

### `stop()`

*   **Описание:** Останавливает двигатели.
*   **Параметры:**  Нет.

### `set_speed(speed)` (только в версии с плавным изменением скорости)

*   **Описание:** Устанавливает скорость двигателей.  Используется для плавного изменения скорости.
*   **Параметры:**
    *   `speed` (int): Скорость двигателя.

**Примечание:** Этот скрипт предназначен для демонстрации управления двигателями.  Для реального управления роботом потребуется интеграция с аппаратным обеспечением (например, с микроконтроллером).
